üéÆ Gamification Management

GET /api/games
Get all games (active by default)
GET /api/games/:id
Get game by ID with all questions
POST /api/games
Create new game
{ "title": "Trivia Challenge", "description": "Test your knowledge", "is_active": 1 }
PUT /api/games/:id
Update game details
DELETE /api/games/:id
Delete game
GET /api/games/:gameId/questions
Get all questions for a game
POST /api/games/:gameId/questions
Create new game question
{ "question_text": "What is the highest altitude?", "options_json": ["35000", "43000", "50000"], "correct_option": 1, "order_index": 0 }
GET /api/games/progress/all
Get all game progress (supports user_id, game_id, question_id filters)
POST /api/games/progress
Record game progress (user answer)
{ "user_id": 5, "game_id": 1, "question_id": 1, "user_answer": "43000", "is_correct": true }
